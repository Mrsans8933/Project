Привет! Это мой мини-проект на пайтоне.
Его суть: Ты вводишь числа и при команде Stop выводит среднее арифметическое число.
Пример:
-----------------
Ввод:
5
4
3
4
5
Stop
Вывод:
4.2
-----------------
Что нужно чтобы запустить?
1. Установленный Python
2. Скачаный файл Arithmetic_mean.py
3. Если у тебя есть какая либо программа для запуска кода (PyCharm/VisualStudio) то просто запустите там
4. Если нету то откройте консоль и введите "python Arithmetic_mean.py"
-----------------
Так же имеется защита от пустого числа (если сразу написать Stop)
-----------------
Что используется?
Только Python3 и его стандартные библеотеки.
