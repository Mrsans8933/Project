Привет! Это мой мини-проект на Python.
Его суть: Это напоминалка которая будет напоминать что либо и когда либо на основе введенных данных.
Пример:
-----------------
Вывод: Привет, я твоя напоминалка! что напомнить?
Ввод: Доделать дела
Вывод: Через сколько напомнить?(В секундах)
Вывод: P.S 1 час = 3600 секунд
Ввод: 3600
Вывод: Отсчет пошел!
Вывод: Напоминание: Доделать дела (три звуковых сигнала)

-----------------
Что нужно чтобы запустить?
1. Установленный Python
2. Скачать файл simple_reminder.py
3. Если у тебя есть какая либо программа для запуска кода (PyCharm/VisualStudio) то просто запустите там
4. Если нету то откройте консоль зайдите в папку со скачанным файлом при помощи "cd путь" и введите "python simple_reminder.py"
