Привет! Это мой мини-проект на пайтоне.
Его суть: Все просто, запускаешь программу, вводишь параметры и на основе введённых данных получаешь пароль!
Пример:
-----------------
Вывод: Привет, это генератор паролей! Введи длину желанного пароля
Ввод: 8
Вывод: Выбери вид пароля: 1 - только цифры; 2 - только буквы; 3 - все вместе
Ввод: 1
Вывод: Ваш пароль:
Вывод: 49355087
-----------------
Что нужно чтобы запустить?
1. Установленный Python
2. Скачать файл Password_generator.py
3. Если у тебя есть какая либо программа для запуска кода (PyCharm/VisualStudio) то просто запустите там
4. Если нету то откройте консоль зайдите в папку со скачанным файлом при помощи "cd путь" и введите "python Password_generator.py"
-----------------
