Привет! Это мой мини-проект на Python.

## Суть

Всё просто: запускаешь программу, вводишь параметры и на основе введённых данных получаешь пароль!

## Пример

Вывод: Привет, это генератор паролей! Введи длину желанного пароля
Ввод: 8
Вывод: Выбери вид пароля: 1 - только цифры; 2 - только буквы; 3 - все вместе
Ввод: 1
Вывод: Ваш пароль: 49355087

## Как запустить

1. Установи Python (если ещё не установлен)
2. Скачай файл `Password_generator.py`
3. Запусти любым способом:
   - **PyCharm / VS Code** — открой проект и нажми Run
   - **Консоль** — перейди в папку с файлом и введи `python Password_generator.py`

## Что используется

- Python 3
- Стандартная библиотека (`random`)
