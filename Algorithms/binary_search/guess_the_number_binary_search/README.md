Привет, это мини-проект по бинарному алгоритму (O(log n))

---

## Суть
Показать работу O(log n) на примере игры «Угадай число»

---

## Пример работы (пользователь загадал 98, диапазон 1–100)

Вывод: Введи максимальное число  
Ввод: 100

Вывод: Загадай число в интервале от 1 до 100  
Вывод: Если число больше — введи +, если меньше — введи -, если компьютер угадал — введи =

Вывод: Твоё число больше 50?  
Ввод: +

Вывод: Твоё число больше 75?  
Ввод: +

Вывод: Твоё число больше 88?  
Ввод: +

Вывод: Твоё число больше 94?  
Ввод: +

Вывод: Твоё число больше 97?  
Ввод: +

Вывод: Твоё число больше 99?  
Ввод: -

Вывод: Твоё число больше 98?  
Ввод: =

Вывод: Попыток 7

---

## Как запустить

- Установленный Python 3
- Скачанный файл `main.py`
- Открыть терминал, перейти в папку с файлом
- Ввести `python main.py`
